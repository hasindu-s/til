## Adding a New TIL Entry

When adding a new TIL entry:

1. Place the file in the appropriate topic folder (e.g., `python/slug.md`). Create the folder if it doesn't exist.
2. Check `README.md` — if the topic folder is not listed under **Categories**, add it.
3. Update the TIL count badge in `README.md` to reflect the new total.
